# 🎓 Ashmika AI Study Buddy

Developed by **Maddikunta Ashmika**

## 📌 Project Description
Ashmika AI Study Buddy is an AI-powered learning assistant built using Streamlit and Google Gemini AI. It helps students understand concepts, generate quizzes, learn through real-life examples, and ask academic questions in simple language.

## ✨ Features
- 📖 Explain Concepts
- 💡 Real-Life Examples
- 📝 Generate Quiz
- ❓ Ask Anything

## 🛠️ Technologies Used
- Python
- Streamlit
- Google Gemini API
- PyPDF
